# Call Sheets

A browser-based call sheet editor for film/TV production. Build and manage daily call sheets with shareable links, CSV import/export, and optional encrypted sharing.

**Live app:** https://wbruntra.github.io/call-sheets/

## Features

- Create and manage multi-day call sheets
- Edit sections, logos, and page breaks inline
- Share a day via compressed URL or encrypted JSON export
- Import/export CSV and JSON
- Tweaks panel for display options

## Development

```sh
bun run dev
```

## Deploy

Builds and publishes to the `gh-pages` branch:

```sh
bun run deploy
```

Hosted at: https://wbruntra.github.io/call-sheets/
